140 medley
==========

This is a micro-framework or a collection of small, helpful utilities for
common javascript tasks.

Size:

* Source:  8.6 kb
* Minified: 821 bytes
* gzipped: 504 bytes

Features:

* templating - t();
* local storage - s();
* bind/unbind events - b();
* create DOM elements - m();
* DOM selector - $();
* Get cross-browser xhr - j();
